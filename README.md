# 🌊 Sistema de Monitoramento de Reservatório

Este é um sistema simples desenvolvido em Python para simular o monitoramento de níveis de água em um reservatório. O objetivo é fornecer alertas visuais coloridos no terminal, facilitando a identificação de riscos.

## 🚀 Funcionalidades

- Exibição de 5 níveis de alerta distintos.
- Uso de cores via biblioteca `colorama` para destacar a criticidade.
- Sistema modularizado com funções e listas.
- Reset automático de estilos para manter a integridade do terminal.

## 📊 Níveis de Monitoramento

| Nível | Situação | Cor |
| :--- | :--- | :--- |
| 1 | Muito baixo (crítico) | Vermelho |
| 2 | Baixo | Amarelo |
| 3 | Médio | Verde |
| 4 | Alto | Ciano |
| 5 | Muito alto (alerta) | Azul |

## 🛠️ Tecnologias Utilizadas

- **Python 3.x**
- **Colorama**: Para estilização do terminal.
